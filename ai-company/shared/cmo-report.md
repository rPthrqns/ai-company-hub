## 1. 타겟 분석

### 1-1. 신규 타겟: 2030 커리어 전환 희망자

| 항목 | 내용 |
|---|---|
| **연령** | 28~38세 |
| **성별** | 여성 비중 높음 (약 65%), 최근 남성 비중 증가 추세 |
| **직무 배경** | 영업·서비스직, CS, 간호사, 학원 강사, 프리랜서, 스타트업 퇴사자 |
| **핵심 니즈** | "안정적인 소득 + 자유로운 스케줄 + 성장 가능성" |
| **고민 포인트** | 보험 설계사에 대한 편견("조달러"), 초반 수입 불안정, 거절에 대한 두려움, 전문성 부족 우려 |
| **정보 획득 채널** | 네이버 검색, 블로그, 유튜브, 맘카페, 당근마켓, 인스타그램 |
| **결정 트리거** | 교육 지원 체계, 선배 설계사 성공 사례, 초기 수입 보장(수습비) 안내, 삼성 브랜드 신뢰도 |
| **전환 시점** | 퇴사 결정 후 1~2주 내 탐색 집중, 주말/저녁 시간대 검색 다수 |

**페인 포인트 매핑:**
- *"보험 하면 거절당할까 봐 무섭다"* → 교육+멘토링 시스템 강조
- *"초반에 돈 벌 수 있나요?"* → 수습 수당+초기 고객 발굴 지원 강조
- *"남들 시선이 부담스럽다"* →  브랜드 프리미엄, 프로페셔널 이미지 강조
- *"육아랑 병행 가능한가요?"* → 유연 근무, 자기 주도적 스케줄 강조

### 1-2. 경력직 타겟: 타사 

| 항목 | 내용 |
|---|---|
| **연령** | 35~55세 |
| **경력** | 3년 이상 설계사 경력, 타사(교보·현대·KB·메리츠 등) 소속 |
| **핵심 니즈** | "더 좋은 컨디션 + 더 많은 고객 + 더 높은 수입" |
| **고민 포인트** | 기존 고객 이동의 어려움, 수습 기간 재수료, 새로운 시스템 적응, 가망고객 제공 여부 |
| **정보 획득 채널** | 설계사 커뮤니티(인스티즈, 네이버 카페), 지인 소개, 헤드헌터/리크루터 |
| **결정 트리거** | 수당 혜택, 수수료 체계, 가망고객 제공, 교육 품질, 사내 문화, 경력 인정 범위 |
| **전환 시점** | 연말 정산 후, 계약 갱신 시기, 불만족도 누적 시 |

**페인 포인트 매핑:**
- *"고객 다 떠나면 어떡하지?"* → 독립 가망고객 제공, 마케팅 지원 강조
- *"처음부터 다시 해야 하나요?"* → 경력별 맞춤 수습, 면제 혜택 강조
- *"수입이 더 늘어나나요?"* →  특유의 풍부한 상품 포트폴리오+수수료 체계 강조

### 2-1. 메인 히어로 섹션 (신규 지원용)

**CTA 버튼 (우선순위 순):**

## 3. 랜딩페이지 구조 기획

### 페이지 정보 아키텍처 (IA)

```
[Header] 로고 + 네비게이션 (신규운영 | 경력운영 | 성공사례 | FAQ | 상담신청)
    │
[Hero] 히어로 섹션 — 타겟별 동적 노출
    │  - 신규 방문자 → 신규 카피
    │  - 경력직 키워드 유입 → 경력 카피
    │  - CTA 버튼 (고정)
    │
[Trust Bar] "  인증 마크 + 숫자 스탯"
    │  - 10,000+ 활동 설계사
    │  - 67년 손해보험 No.1
    │  - 연간 신규 입사자 3,000+명
    │
[Section 1] "이런 분께 추천합니다"
    │  - 신규: 아이콘 4개 (커리어 전환, 육아·일 병행, 소득 상승, 자기계발)
    │  - 경력: 아이콘 4개 (수입 UP, 고객 풀, 시스템, 성장)
    │
[Section 2] "왜 인가?" (3개 핵심 포인트)
    │  1) 브랜드 파워 (시장점유율, 인지도)
    │  2) 교육·지원 시스템 (멘토링, 가망고객 제공)
    │  3) 수입 체계 (수습수당, 수수료 구조)
    │
[Section 3] "성장 과정" (타임라인 형태)
    │  1개월: 온보딩 교육 → 2~3개월: 멘토와 함께 실전 → 6개월: 독립
    │  1년: 평균 월 500만+ → 3년: 연 8,000만+ (스케일 인포그래픽)
    │
[Section 4] 성공 사례/후기 (캐러셀 형태)
    │  - 신규 2건, 경력 1건
    │  - 사진 + 이름(이니셜) + 전 직업 + 현재 수입 + 인터뷰 발췌
    │
[Section 5] 지원 혜택 요약 (카드 그리드)
    │  - 수습수당 | 자격증 지원 | 가망고객 제공 | 교육비 지원 | 스터디 그룹 | 워크숍
    │
[Section 6] FAQ (아코디언 형태)
    │
[Sticky CTA] 하단 고정 신청 폼 또는 버튼
    │
[Footer] 회사 정보 | 개인정보처리방침 | 고객센터
```

### 상세 섹션 스펙

#### Section 1: "이런 분께 추천합니다"
- **형태:** 2x2 아이콘 카드 그리드
- **신규용 내용:**
  1. 🔄 "지금 하는 일에 성장 한계를 느끼시나요?"
  2. 💰 "노력한 만큼 정당하게 보상받고 싶으신가요?"
  3. ⏰ "내 시간을 내가 결정하는 일을 원하시나요?"
  4. 📈 "전문가로 성장하는 커리어를 원하시나요?"

#### Section 2: "왜 인가?"
- **형태:** 3개 탭 또는 스크롤 섹션
- **포인트 1 — 브랜드:** 시장점유율 그래프, 브랜드 인지도 수치, "삼성" 이름값
- **포인트 2 — 시스템:** 교육 커리큘럼 인포그래픽, 멘토링 매칭 시스템, 가망고객 제공 프로세스
- **포인트 3 — 수입:** 수입 성장 커브, 수수료 구조 다이어그램, 실제 수입 범위

#### Section 6: FAQ (10개 항목)
1.  자격증이 없어도 지원 가능한가요?
2. 수습기간 동안 수입은 어떻게 되나요?
3. 초보자도 충분히 성공할 수 있나요?
4. 근무 시간과 형태는 어떻게 되나요?
5. 경력직 전입 시 기존 고객은 어떻게 되나요?
6. 교육은 어떻게 진행되나요?
7. 가망고객은 어떻게 제공되나요?
8. 퇴사 후 재입사도 가능한가요?
9. 필요한 서류와 지원 절차는?
10. 지원 후 합격까지 얼마나 걸리나요?

### 5-2. 경쟁사 랜딩페이지 공통 문제점

1. **원 페이지 길이 과도** — 스크롤이 너무 길어 이탈률 높음
2. **타겟 분리 미흡** — 신규와 경력을 하나의 페이지에 섞어 배치
3. **수치 부재** — "높은 수입"이라는 추상적 표현만 사용, 구체적 데이터 없음
4. **후기 신뢰도 낮음** — 실명 없는 후기, 범용적 인터뷰만 게재
5. **CTA 모호** — "자세히 보기", "문의하기" 등 불명확한 버튼
6. **모바일 UX 열악** — PC 위주 레이아웃, 터치 타겟 작음

### 5-3.  차별화 포인트 (USP)

#### ① 브랜드 프리미엄 (가장 강력한 무기)
- **"삼성"이라는 이름**만으로 고객 신뢰 확보 가능
- 시장점유율 30%+ 독보적 1위 (2025년 기준)
- **카피 포인트:** *"고객이 문을 열어주는 3초,  이름값이 대신합니다."*

#### ② 상품 포트폴리우스 광범위
- 손해보험(자동차, 화재, 실손) + 생명보험(삼성생명) + 3자타사 상품까지
- **카피 포인트:** *"한 설계사가 모든 보험을, 하나의 상담으로 해결합니다."*

#### ③ 가망고객 제공 시스템
- 경력직 전입 시 독립 가망고객 제공
- 신규 입사자 대상 DM/텔레마케팅 지원
- **카피 포인트:** *"고객을 찾는 시간은 우리가, 고객과 만나는 시간은 당신이."*

#### ④ 교육 인프라
-  아카데미 (전국 교육장 15개+)
- 온라인 교육 플랫폼 (언제든 학습 가능)
- **카피 포인트:** *"대기업 수준의 교육 인프라가 설계사의 성장을 가속합니다."*

#### ⑤ 수입 보장 체계
- 수습수당 (타사 대비 상위권)
- 안정 수당 (최저 보장 수입)
- **카피 포인트:** *"첫 달부터 빈손이 아닙니다."*
