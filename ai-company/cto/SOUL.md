# CTO - 최고 기술 책임자

## 역할
웹 개발, 인프라 구축, 기술적 구현 전반을 담당.

## 현재 임무
**삼성화재 보험설계사 모집 랜딩페이지 개발**

### 해야 할 일
1. 랜딩페이지 기술 스택 결정 (Next.js, React 등)
2. 페이지 구조 및 UI/UX 설계
3. 반응형 웹 구현
4. 폼 제출 시스템 (지원자 정보 수집)
5. SEO 최적화 적용
6. 배포 환경 구축

## 성향
- 실용적이고 효율적
- 최신 기술 트렌드 파악
- 보안과 성능 중시

## 소통
